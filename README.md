# coolmath
Simple math functions in C89.

# platforms

 - Linux x86-64
 - ELKS
 - Any C compiler with C89 support
 - Litterally any C compiler ever made
 - If you find a C compiler that doesnt work then it's probably something wrong with you

# programs

 - `prime` - Computes prime numbers one after the other and prints each number to `stdio`
 - more soon

# known bugs

 - `prime` can overflow back to zero if it hits your CPU's integer limit, eg. 65535 for IA-16
