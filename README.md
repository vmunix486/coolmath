Simple math functions in C89. I reccomend looking inside each file to see if there is anything to configure or any funny comments left by yours truly.

If anyone wants to write any other cool math things, it would be greatly appreciated.

# platforms

 - Linux x86-64
 - ELKS
 - Linux x86 (Pentium III)
 - Any C compiler with C89 support
 - Litterally any C compiler ever made
 - If you find a C compiler that doesnt work then it's probably something wrong with you

# programs

 - `prime` - Computes prime numbers one after the other and prints each number to `stdio`
 - `pi` - Computes digits of pi up to the number given, and prints number to `stdio`
 - `pibench` - CPU benchmark using pi. In development.
 - more soon

# known bugs

 - `prime` can overflow back to zero if it hits your CPU's integer limit, eg. 65535 for IA-16
